== Description ==

You want the Contact Form 7 response message when you push submit in a popup colorbox window? This small plugin does just that. 

You need Lightbox Plus Colorbox and Contact Form 7 installed.

This plugin does not put an entire contact form inside a colorbox popup but just the response message when you click the Submit Button.

== Installation ==

1. Make sure you have Lightbox Plus Colorbox and Contact Form 7 installed.
2. Download the file to install and activate.
3. Done! Just remember to keep the plugins up to date!

== Frequently Asked Questions ==

= Why I cannot activate CF7RCP =

This plugin requires Contact Form 7 and Simple Colorbox or Lightbox Plus Colorbox. Install them first and then try active it again.

= Why my message is unstyled in the Colorbox popup? =

You have an older version of Lightbox Plus Colorbox. Updating it should fix the problem.
If not, try another plugin that has colorbox js library like : Simple Colorbox(make sure to deactivate Lightbox Plus Colorbox).

= How do I know if this is for me? =

This plugin doesn't put an entire contact form inside a colorbox popup but just the response message when you click the Submit Button.
